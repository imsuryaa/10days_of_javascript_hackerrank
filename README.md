# 10days_of_javascript_hackerrank

## Day 1

- Calculate the area and perimeter of rectangle

```
function getArea(length, width) {
    let area;
    area = length * width;
    return area;
}
function getPerimeter(length, width) {
    let perimeter;
    perimeter = 2 * (length + width);
    return perimeter;
}
```
